# GoodShowHQ Home Assistant Add-ons

[![Add repository to Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fa7medkamel%2Fgoodshowhq-ha-addons)

Add-ons:

- **GoodShowHQ** — sign in to your kid's YouTube once from Home
  Assistant, then keep their feed monitored and your approved
  actions applied around the clock. See the add-on's DOCS tab for
  setup.

To install manually: Settings → Add-ons → Add-on Store → ⋮ →
Repositories → add this repo's URL.
